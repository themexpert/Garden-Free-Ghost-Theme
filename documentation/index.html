Theme Installation

This documentation is intended to help you properly install and configure Garden for your Ghost blog. Check the following step-by-step instructions on how to perform a manual Ghostion theme installation.
  1. Unzip garden.zip and copy the folder into your Ghost theme directory /content/themes .
  2. At the moment, Ghost won't notice that you've added a new folder to content/themes directory, so you'll need to restart it.
  3. After restart, login to your Ghost dashboard and navigate to Settings - General.
  4. Select garder name in the Theme options dropdown.
  5. Click Save. That's all.
  6. Visit the frontend of your blog and marvel at the new Ghostion theme.

How to make any section?

For an example you want to create a section name Products then please go throung following instructions

STEP 01

Go to top-header>.main-nav section

 

STEP 02

Make a list element like 

<li><a data-scroll href="#products" > <span data-hover="product">Products</span></a></li>

STEP 03

Now you have to place section where you will put this section content. As example I have added link after skill section so go to bottom of skill section make a section like

<section id="products" class="products">

<div class="section-title">

    <h2>Products</h2>
    <p> Your text here<span>desert raisin</span> taro wakame pea sprouts. Broccoli rabe courgette rock <span>melon cress </span> epazotebell</p>

</div>


<div class="container">

    <div class="row">

    <!-- You can use any column here as like as Bootstrap column stytem -->

        <div class="col-md-4 wow fadeIn ">

         Column

        </div>

        <div class="col-md-4">

            Column

        </div>

        <div class="col-md-4 wow fadeInRight ">

            Column

        </div>

        </div>

    </div>

</div>

</section>
No need to add any link in nav if you want to create any section that will not a part on top navigation. And then no need to use id="" and no need to use appear class

It is very easy just use just go to http://daneden.github.io/animate.css/ this site and choose an animation and add animation name by following way in any elemenet:

Themes And Layout

This template have 5 predefine themes. All themes files have in css/themes folder. This template have following themes.

If you want to use any theme you have to just replace theme file in following line:

<link href="css/main.css" type="text/css" rel="stylesheet">
Now it is default theme. If you want to use blue theme then just replace path by following way. Your theme will be change instantly.
<link href="css/themes/blue.css" type="text/css" rel="stylesheet">
